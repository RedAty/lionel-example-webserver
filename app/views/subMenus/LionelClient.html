LionelClient

A piece of backend everywhere

Client Side Library for Lionel

LionelClient is an object to communicate backend in /app/server/ folder, call Rest API endpoints, and handle some basic request from the Developer to help and speed up making a good project.

Functions

LionelClient.call(methodName, ..parameters[OPTIONAL], callback[OPTIONAL])

Call function make a Rest API endpoint to receive result from a Lionel method connected to the specific call.
LionelClient.call('getResult') function call a Lionel Method on Server with name "getResult" and answer with the result of that function.
More details in the Methods Page

LionelClient.getPublicJS(jsName)

We can ask server to load a javascript file/library to the browser Async anytime with this event. Every javascript can be loaded into a HTML script from the /app/public/js/ directory.
Usage: LionelClient.getPublicJS('bootstrap').then(()=>{console.log('Loaded /app/public/js/bootstrap.js')})

LionelClient.navigate(href)

We can navigate in pages with this function instead of using hyperlinks. This function does not refresh the page and load every dependencies again, just refresh the HTML content and call the onRendered javascript for the specific page. If you open the Index page and navigate to About page with this function and go back to Index, the last time will much faster because the javascript with dependencies are loaded from cache and called from there.

LionelClient.mongodb(queryType, query);

If the server is connected to a mongoDB instance we can use the API and communicate with mongoDB.
If we call LionelClient.mongodb() we will get some extra function what is helpful for beginners in mongoDB like

  • LionelClient.mongodb().add(collection,filter)
  • LionelClient.mongodb().delete(collection,filter)
  • LionelClient.mongodb().find(collection,filter)
  • LionelClient.mongodb().listCollections()
  • LionelClient.mongodb().update(collection,filter,details)
LionelClient.mysql(queryType, query);

If the server is connected to a MySQL instance we can use the API and communicate with mySQL server.
If we call LionelClient.mysql() we will get some extra function what is helpful for beginners in SQL like

  • LionelClient.mysql().delete(table,filter)
  • LionelClient.mysql().select(table,filter)
  • LionelClient.mysql().selectAll(table)
  • LionelClient.mysql().update(table,filter,data)
  • LionelClient.mysql().listTables()
  • LionelClient.mysql().removeTable(table)
