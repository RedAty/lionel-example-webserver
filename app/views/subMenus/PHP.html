Export Server as PHP Web Service (Beta)

If we need to upload our application to a common Web Server for our domain, then we need to rend to rent a Javascript Server to run our Node JS instance or we can just export the whole web page.
This means we export our Router to PHP and HTML files, so it is not 100% sure that every functionality will be working, but the following methods are included in Lionel-PHP version:

  • Every LionelClient function except LionelClient.call and LionelClient.mongodb (with experience in PHP it is possible to upgrade)
  • SQL compatibility

For more details check the Roadmap.
