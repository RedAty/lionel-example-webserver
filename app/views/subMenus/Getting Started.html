Getting Started

Get started with Lionel App

First Steps

  • We need to download the first template of Lionel-App from GitHub or from the website
  • Download and install Node Js with npm from here
  • Open Lionel App folder and run "npm install" command to keep update your npm dependencies

Check main server files

First of all to have a fully functional Javascript Web Application you need to open a server.js in the main directory. The file should contain something like this:

'use strict';
const {app} = require("./app");
const {initLionelServer} = require("lionel-app/bin/initLionelServer");
const { jsFolder, viewFolder, libFolder, port, liveUpdate, mainDirectory, separator, appData, appName} = require("./constants");

/**
* Start HTTP/HTTS Server
* @type {Array|*}
*/
initLionelServer(port,{
appData: appData,
appName: appName,
lib:libFolder,
view:viewFolder,
js:jsFolder,
debug:true,
mainDirectory:mainDirectory,
separator:separator,
requestListener:app
});
/**
* Initialize routes for the webPage
*/
require('./app/routes.js');

/**
* Start Server Backend codes
*/
require('./app/server/index');

For this you will need a constants.js file next to the server.js. In this files you can edit variables what connected to your Application, like Application Name, or you can edit folder structure of Lionel App too.
