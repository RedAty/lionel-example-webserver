Lionel Methods

Communication between client and server

From the client side we can call functions on the server and work with the results of them. On the client we can use the LionelClient.call() functions for this tasks to call functions in the Lionel.methods() on the server side.

app/views/index.js

LionelClient.call('getSampleText',function (error, result) {
console.log(error,result)
});
LionelClient.call('getSampleJSON').then(function (result) {
console.log(result);
}).catch(function (error) {
console.warn(error);
});
app/server/index.js

Lionel.methods({
/**
* This can be called with Lionel.call('getSampleText',..args,callback) returns Promise
* @returns {string}
*/
getSampleText: function () {
return 'Text'
},
/**
* This can be called with Lionel.call('getSampleJSON',..args,callback) returns Promise
* @returns {string}
*/
getSampleJSON: function () {
return {
status: 'JSON value'
}
},
});
